# S-Maker

Мой помощник для разработки игр на Pixi.js ;)

[Демо](https://selaxgit.github.io/s-maker)

Фреймворк: Angular

Хранение данных: IndexedDB

### Библиотеки

- Angular material
- Pixi.js
- jszip
- @ngrx/component-store

### Установка и запуск

```shell
$ npm install
$ npm run start
```

### Сборка Electron

```shell
$ ./build-electron.sh
```
